# 🧬 DSB深度人格映射系统

> **"在崩坏的世界里，保持真实的懒惰。"**

DSB (Deep Soul Blueprint)是一款基于前端原生技术栈开发的**沉浸式人格鉴定程序**。不同于传统 MBTI 这种充满“正能量伪装”的测试，DSB 专注于捕捉你潜意识中的社恐、摸鱼、脑洞与反矫情成分。

## 🎮 如何使用

1.  克隆或下载 `index.html` 文件。
2.  在任何现代浏览器（Chrome, Edge, Safari, Firefox）中直接打开。
3.  依据直觉回答 30 道行为逻辑题。
4.  获取你的 DSB 映射结果，并查看四个维度的原始数据。

## 📝 题目逻辑说明

程序通过 `bank` 数组存储题库，每道题目关联一个维度 ID：
* `c: 0` -> 脑洞维度
* `c: 1` -> 社恐维度
* `c: 2` -> 咸鱼维度
* `c: 3` -> 解构维度

用户选择的程度（从 +2 到 -2）将实时累加至对应维度的向量池中，最终通过 `Math.max` 锁定主导人格。
